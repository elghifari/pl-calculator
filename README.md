**P&L Cashflow Calculator**

A mobile-first simple calculator to project monthly and annual profit for a barbershop, including P&L breakdown and per-barber earnings estimates.

- Quick scenarios: Low / Mid / High / Custom
- Interactive sliders for all parameters (cut price, number of barbers, base pay, bonus, overhead)
- Monthly P&L summary and annual projection
- Per-barber earnings breakdown
- Sticky bar always showing the key numbers
